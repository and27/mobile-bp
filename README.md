# MobileBP - React Native (Expo)

Frontend mobile app built with React Native, Expo, and TypeScript.

## Requisitos

- Node.js 22+
- npm 10+

## Instalación

```bash
npm install
```

## Ejecutar proyecto

```bash
npm run start
```

Opciones rápidas:

- `npm run android`
- `npm run ios`
- `npm run web`

## Pruebas unitarias

Ejecutar pruebas:

```bash
npm run test -- --watch=false --runInBand
```

Ejecutar cobertura:

```bash
npm run test:coverage
```

Thresholds de cobertura configurados:

- Branches: `70%`
- Functions: `70%`
- Lines: `70%`
- Statements: `70%`
